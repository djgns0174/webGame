- battle 모드

공격스킬

area : 상대 탱크 크기 증가
chance : 2회 기회
damage : 공격력 2배
big : 본인 포탄 크기 증가

방어스킬

small : 다음 턴에 상대 미사일 크기 감소
decrease : 다음 턴에 내 탱크 크기 감소
move : 이동거리 2배 증가
hp : 체력 50 회복

- target 모드

area : 목표물 크기 증가
chance : 2회 기회
damage : 점수획득 2배
big : 본인 포탄 크기 증가
